# Terraform Infrastructure

This directory contains the Terraform scaffolding for the Aegis-Deploy platform.

The goal of this structure is to demonstrate how production cloud infrastructure can be organized using modular Infrastructure-as-Code.

Modules include:

- Network (VPC and subnets)
- Load Balancer (ALB and routing rules)
- ECS Services (Fargate deployments)
- Observability (CloudWatch logs and metrics)

This scaffold is designed to illustrate architecture structure rather than provide a full production implementation.
